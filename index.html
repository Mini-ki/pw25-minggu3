WELCOME TO
SIMPELAH!

SIMPELAH is an innovative solution to Indonesia’s growing waste problem. This app leverages technology to boost public awareness and engagement in waste sorting, making waste management more efficient and sustainable.

LET'S SCANNING!!

SIMPELAH provides a Waste Scanning feature that allows users to scan waste and instantly obtain information about it, including its type and proper disposal methods. By using Waste Scanning, users can make more informed decisions about waste management, contributing to a cleaner and more sustainable future.
